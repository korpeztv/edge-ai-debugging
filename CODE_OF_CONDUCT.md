# Code of Conduct

## Our Pledge

We want this project to be a welcoming place for everyone. By participating in this repository, you agree to treat other people with respect and professionalism.

## Expected Behavior

Examples of behavior that helps maintain a healthy community:

- Be constructive, respectful, and direct.
- Focus feedback on the content of the contribution, not the person making it.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions relevant to the project and its prompt-evaluation goals.

## Unacceptable Behavior

Examples of behavior that does not belong in this project:

- Harassment, threats, insults, or discriminatory language.
- Deliberate trolling, personal attacks, or repeated hostility.
- Sharing private, sensitive, or identifying information without permission.
- Disrupting discussion or review threads in ways that block collaboration.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, review comments, commit messages, and any other communication associated with this repository.

## Reporting Problems

If you experience or witness behavior that violates this code of conduct, report it by opening a private security or repository contact path if available, or by notifying the repository maintainers through the usual project communication channels.

## Enforcement

Maintainers may remove content, lock discussions, or restrict participation for conduct that violates this policy.

## Acknowledgment

Participation in this repository means you agree to follow this code of conduct.
